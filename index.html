HTML

  • HTML is the basic framework for a website
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE declaration lets the browser know what to expect on the page.
  • HTML attributes are always defined in the beginning element not the closing element
  • The head element can use the following meta tags:title, style, base, link, script, and meta
  • Elements that hold other elements inside are called parent elements and the elements nested inside child elements
  • The h1-6 elements are used for headers and the h1 element must only be used once
  • The p element is for bodies of text
  • ul element is used for unordered lists
  • ol element is used for ordered lists
  • li element is used for list items
  • img element is contains info on the images displayed on the website
  • br element is used for creating breaks between lines
  • a element is used for creating links to other or the same webpage

CSS

  • Cascading Style Sheets (CSS), which is a programming language used by developers to define how webpage content is presented to users. Whereas HTML strictly handles the structure and content of a webpage, CSS defines the colors, aesthetics, visual layout, and organization. We use CSS to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS: inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration color: blue; includes the CSS property color and the value for that property, blue
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained. DRY can greatly reduce the overall size and complexity of a codebase.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
